# Sway Client

A frontend built with React, Shadcn/ui, and more. For more, please view the README.md in the root of this project.

---

## Core Dependencies

**Production**


- React
- TypeScript
- React Router
- Tanstack Query
- Zustand
- Axios
- TailwindCSS
- Shadcn/ui
- Recharts
- Socket.io-client


---

## Installation

```bash
git clone git@github.com:jordansmalls/sway.git
cd client
bun install
bun run dev
```
